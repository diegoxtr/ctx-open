# CTX - Release 1.0.3

Fecha de release: 2026-04-14

Versión:

- `1.0.3`

Resumen:

- Patch estable para CTX 1.0.
- Congela la baseline actual después del pase de legibilidad del grafo del viewer, el endurecimiento del preflight basado en runbooks y el primer flujo GitHub-native de live demo y entrega pública.

Highlights:

- El viewer ahora enfoca `Working` en la línea táctica inmediata, resalta el lineage completo del nodo seleccionado y renderiza columnas y aristas largas del grafo de forma más legible.
- `ctx preflight --operation ...` entrega guidance de preflight más fuerte para closeout, publish local, validación del viewer y recuperación de locks.
- El scaffolding de live demo con GitHub Codespaces y la landing estática ya forman parte de la baseline de release.
