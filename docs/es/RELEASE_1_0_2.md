# CTX - Release 1.0.2

Fecha de release: 2026-04-14

Version:

- `1.0.2`

Resumen:

- Release patch estable de CTX 1.0.
- Congela la linea base actual despues de que convergieran en un mismo estado coherente el rediseño del rail derecho del viewer, las superficies compactas de memoria operativa y la paridad de guias en tres idiomas.

Highlights:

- El viewer ahora expone `Origin`, `Playbook` y `Hypotheses` como tabs aisladas del rail derecho, con side tab rails para la navegacion colapsada.
- `OperationalRunbook` y `CognitiveTrigger` ahora son superficies versionadas de primer nivel dentro del repositorio.
- El set canonico de guias bajo `docs/` quedo alineado entre ingles, español y chino.
