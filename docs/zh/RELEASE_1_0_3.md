# CTX - Release 1.0.3

Release date: 2026-04-14

Version:

- `1.0.3`

Summary:

- CTX 1.0 的稳定补丁版本。
- 冻结当前基线：包含 viewer 图可读性改进、更强的 runbook preflight，以及首个 GitHub 原生 live demo / public delivery 流程。

Highlights:

- viewer 现在会把 `Working` 聚焦到当前直接战术谱系，支持对所选节点的完整 lineage 高亮，并让列布局和长边路由更易读。
- `ctx preflight --operation ...` 为 closeout、本地发布、viewer 验证和 lock 恢复提供更强的 runbook 预检指导。
- GitHub Codespaces live demo 脚手架与静态 landing 现在都属于当前 release baseline 的一部分。
