# Bootstrap Agriculture Demo V2

This example is the second-pass version of the agricultural bootstrap validation.

It repeats the same class of test as `bootstrap-agriculture-demo`, but with a different design target:

- do not reward early synthesis
- preserve competing interpretations as parallel provisional hypotheses
- evaluate whether CTX can hold contradiction without forcing premature coherence

Primary source strategy:

- use the contradiction-heavy agricultural text again
- keep the same domain difficulty
- change the bootstrap interpretation strategy, not the problem domain

Artifacts for this version:

- [V2_PLAN.md](C:/sources/ctx-open/examples/bootstrap-agriculture-demo-v2/V2_PLAN.md)

In `ctx-open`, this folder is published as a design-and-testing pack.
If you want to replay the case, initialize it locally as its own CTX repository so the `v2` experiment can evolve independently from the closed `v1` demo.
