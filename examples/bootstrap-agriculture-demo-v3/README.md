# Bootstrap Agriculture Demo V3

This example is the implementation-validation repo for the coexistence-first bootstrap strategy.

It exists after:

- `bootstrap-agriculture-demo` proved that CTX review can manually recover multiple hypotheses
- `bootstrap-agriculture-demo-v2` proved that the current installed bootstrap still collapses the contradiction-heavy article into one literal hypothesis

The purpose of `v3` is different:

- validate the implementation change
- test the same contradiction-heavy article again
- confirm that multiple hypotheses survive directly from the bootstrap flow

Expected success condition:

- `microbiota-first`
- `microclimate-first`
- `complex interaction`

should appear as separate provisional hypotheses instead of collapsing into one synthesized line by default.

Artifacts for this version:

- [V3_PLAN.md](C:/sources/ctx-open/examples/bootstrap-agriculture-demo-v3/V3_PLAN.md)

In `ctx-open`, this folder is published as an implementation-validation pack.
If you want to replay the case, initialize it locally as its own CTX repository so the implementation result can be measured independently from `v1` and `v2`.
