# Viewer Demo

This folder is a compact repository for validating CTX Viewer behavior.

Use it when you want to inspect:

- branch switching
- timeline lanes
- commit history rendering
- graph focus behavior

Unlike the `examples/ctx/*` repositories, this folder is not meant to be a narrative product demo. It is primarily a viewer validation workspace.

To inspect it in the viewer:

```powershell
dotnet run --project C:\sources\ctx-public\Ctx.Viewer
```

Then load:

```text
examples\viewer-demo
```
